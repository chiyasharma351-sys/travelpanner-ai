# **App Name**: TravelGenAI

## Core Features:

- Itinerary Generation: Generate detailed day-wise travel itineraries based on user input parameters such as destination, dates, budget, and number of travellers.
- Itinerary Personalization Tool: Use generative AI as a tool to intelligently personalize itineraries, incorporating previous travel data to avoid repetitions and align with user preferences. Reasoning by the tool will lead to subtle, but effective variations from previously visited locations.
- Structured Itinerary Display: Display itineraries in a clear, day-wise format, including morning, afternoon, and evening activities, commute information, food recommendations, and relevant notes.
- Travel Tips: Provide helpful travel tips specific to the destination, covering weather, safety, and transportation.
- Parameter Input: Allow users to input travel parameters such as destination, dates, budget type (economical, standard, luxury), and the number of travellers.

## Style Guidelines:

- Primary color: Calm blue (#5DADE2) to evoke trust and serenity.
- Background color: Soft light blue (#EBF5FB), offering a gentle contrast.
- Accent color: Warm yellow (#F4D03F) to highlight important information and calls to action.
- Body and headline font: 'PT Sans', a humanist sans-serif to give a modern yet inviting feel.
- Use clear, universally recognized icons for weather, safety, and transportation.
- Day-wise itinerary segments should be clearly separated with sufficient whitespace. Each segment should emphasize an activity category.
- Subtle transitions when displaying itinerary details or tips.
